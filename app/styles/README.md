# Styles Directory

This directory contains additional CSS modules and style utilities for the Flyx application.

## Structure

- `globals.css` - Global styles with Tailwind directives and custom CSS variables
- Additional component-specific CSS modules can be added here as needed

## Usage

Import styles in components:
```tsx
import styles from '@/styles/component.module.css'
```

## Tailwind Configuration

The project uses Tailwind CSS with a custom futuristic theme. See `tailwind.config.ts` for configuration.
