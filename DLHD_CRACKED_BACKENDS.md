# DLHD Cracked Backends - January 2026

## Summary

After extensive reverse engineering, here are the DLHD backends that can be cracked with pure fetch:

### ✅ WORKING BACKENDS (No encryption, no complex auth)

#### 1. moveonjoy.com (Player 6)
- **Path**: DLHD /player/ → tv-bu1.blogspot.com → moveonjoy.com
- **Auth**: None required
- **Encryption**: UNENCRYPTED
- **Channels**: ~35 channels mapped
- **Example**: `https://fl2.moveonjoy.com/ESPN_2/index.m3u8`

#### 2. cdn-live-tv.ru (Player 5)
- **Path**: DLHD /casting/ → ddyplayer.cfd → cdn-live-tv.ru
- **Auth**: Token in URL (extracted from HUNTER-obfuscated script)
- **Encryption**: UNENCRYPTED
- **Channels**: Many sports channels
- **Example**: `https://cdn-live-tv.ru/api/v1/channels/us-espn/index.m3u8?token=xxx`

#### 3. lovecdn.ru/popcdn.day
- **Path**: DLHD /player/ → lovecdn.ru → popcdn.day → beautifulpeople.lovecdn.ru
- **Auth**: Token in URL (generated by popcdn.day)
- **Encryption**: UNENCRYPTED
- **Channels**: ESPN, ESPN2, FOXSPORTS1, SPORT5LIVE, etc.
- **Example**: `https://beautifulpeople.lovecdn.ru/ESPN/index.m3u8?token=xxx`

### ❌ NOT CRACKED (Complex obfuscation)

#### 4. justembeds.xyz (Player 4)
- **Path**: DLHD /plus/ → daddyliveplayer.shop → justembeds.xyz
- **Problem**: Heavily obfuscated JavaScript with custom encoding
- **Status**: Stream URL is encoded in `window['ZpQw9XkLmN8c3vR3']` variable
- **Channels affected**: 52, 53, and others without moveonjoy/cdn-live-tv mapping

#### 5. epicplayplay.cfd
- **Status**: DEAD (DNS not found)
- **Channels affected**: 565, 568, 770 (Polish channels)

### ❌ REQUIRES JWT (topembed.pw)

#### 6. dvalna.ru (Player 3)
- **Path**: DLHD /watch/ → topembed.pw → dvalna.ru
- **Auth**: JWT + PoW (Proof of Work)
- **Encryption**: AES-128 encrypted segments
- **Problem**: JWT must be fetched from topembed.pw, keys require valid JWT

## Implementation

The tv-proxy.ts should try backends in this order:
1. moveonjoy.com (fastest, no auth)
2. cdn-live-tv.ru (token auth, but simple)
3. lovecdn.ru/popcdn.day (token auth)
4. Skip justembeds.xyz (too complex)
5. Skip dvalna.ru (requires topembed JWT)

## Code to Extract Streams

### moveonjoy.com
```javascript
// Fetch DLHD /player/ page
const dlhdRes = await fetch(`https://dlhd.link/player/stream-${channel}.php`);
const blogspotMatch = dlhdRes.text.match(/src=["'](https?:\/\/[^"']*blogspot[^"']+)["']/i);
const blogRes = await fetch(blogspotMatch[1]);
const m3u8Match = blogRes.text.match(/https:\/\/fl\d+\.moveonjoy\.com\/[^"'\s]+\.m3u8/);
// m3u8Match[0] is the stream URL
```

### cdn-live-tv.ru
```javascript
// Fetch DLHD /casting/ page
const dlhdRes = await fetch(`https://dlhd.link/casting/stream-${channel}.php`);
const ddyMatch = dlhdRes.text.match(/src=["'](https?:\/\/ddyplayer\.cfd[^"']+)["']/i);
const ddyRes = await fetch(ddyMatch[1]);
// Decode HUNTER obfuscation
const hunterMatch = ddyRes.text.match(/\}\s*\(\s*"([^"]+)"\s*,\s*(\d+)\s*,\s*"([^"]+)"\s*,\s*(\d+)\s*,\s*(\d+)\s*,\s*(\d+)\s*\)\s*\)/);
const decoded = decodeHunter(hunterMatch[1], hunterMatch[3], parseInt(hunterMatch[4]), parseInt(hunterMatch[5]));
const m3u8Match = decoded.match(/https:\/\/cdn-live-tv\.ru\/[^"'\s]+\.m3u8[^"'\s]*/);
```
